This is a sample document about machine learning pipelines.

## Overview

A machine learning pipeline automates the workflow from raw data to a trained model.
It includes stages for data ingestion, preprocessing, feature engineering, model
training, evaluation, and deployment.

## Data Ingestion

The first stage of any ML pipeline is getting data into the system. Data sources
can include databases, file systems, APIs, and streaming platforms.

## Feature Engineering

Feature engineering transforms raw data into features that better represent the
underlying problem. This includes normalization, encoding categorical variables,
and creating interaction features.

## Model Training

Model training uses the engineered features to learn patterns from historical data.
Common algorithms include linear regression, decision trees, random forests, and
neural networks.

## Deployment

Once trained and validated, models are deployed to production where they serve
predictions to downstream applications.
