Model Evaluation Metrics

Choosing the right evaluation metric is essential for assessing model performance. Different problems require different metrics.

Classification Metrics

  • Accuracy: Proportion of correct predictions. Best for balanced datasets.
  • Precision: True positives divided by all positive predictions.
  • Recall: True positives divided by all actual positives.
  • F1 Score: Harmonic mean of precision and recall.

Regression Metrics

  • MAE: Mean Absolute Error — average of absolute differences.
  • MSE: Mean Squared Error — penalizes large errors more heavily.
  • RMSE: Root Mean Squared Error — interpretable in original units.
  • R-squared: Proportion of variance explained by the model.
